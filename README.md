# Kobold Crystal Mine (Crystal Clicker)

Inspired by Cookie Clicker by Orteil

## How to run it

-   run `git clone https://github.com/DavKub/crystal-clicker.git`
-   run `yarn install`
-   run `yarn start`
-   enjoy the "game"
